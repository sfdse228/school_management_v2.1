# 🏫 Система управления онлайн-школой v2.1

Консольное приложение для управления студентами, преподавателями и курсами с аналитикой, визуализацией и экспортом данных.

## 📋 Возможности

### Управление данными
- Студенты (имя, email)
- Преподаватели (имя, специализация)
- Курсы (название, тема, модули, уроки)

### Функциональность
- Зачисление студентов на курсы
- Назначение преподавателей
- Выставление оценок (0-100)
- Домашние задания с оценками
- Завершение курсов

### Аналитика
- Отчёт по студенту (активные/завершённые курсы, оценки)
- Отчёт по преподавателю (курсы, успеваемость)
- Топ студентов по успеваемости
- Статистика по курсу

### Визуализация
- Графики успеваемости студента
- Распределение оценок
- Оценки за домашние задания

### Экспорт
- CSV-отчёт по студенту
- CSV-отчёт по преподавателю
- CSV-отчёт по курсу

### Хранилище
- Автосохранение в JSON
- Загрузка при запуске
- Восстановление всех связей

## 🚀 Установка и запуск

```bash

# 2. Установить зависимости
